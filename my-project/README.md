# Install dependencies

To plot your programs, you will need graphviz

```shell
sudo apt update
sudo apt install graphviz
```

# Install your project as a library

```shell
cd my-project
uv venv
uv pip install .
```

# Train your application

```shell
cd my-project
python3 scripts/train.py
```

# Serve your API

```shell
cd my-project
docker compose up
```
